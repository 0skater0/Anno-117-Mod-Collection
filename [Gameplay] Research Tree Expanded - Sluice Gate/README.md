# EN – Description

**Expand the research tree with six new Sluice Gate technologies that naturally ramp up drainage capacity and upkeep efficiency, on top of a broader drainage width from the very start.**  
This mod adds a small research branch in the Economy tech tree that improves Sluice Gates in Albion instead of buffing them outright, while the basic coverage of drainage channels is increased globally as soon as the mod is active.

## Features
- **Six new research projects** dedicated to Sluice Gates and drainage channels.
- **More drainage channels per Sluice Gate**: early upgrades increase pipe capacity so each gate can support additional drainage channels.
- **Wider drainage coverage from the start**: the base spread of drainage channels is increased globally by this mod, independent of research, so coverage feels better right away.
- **Lower upkeep on the Sluice Gate itself**: later upgrades reduce the maintenance cost of the Sluice Gate building that drives all attached channels.
- **Alternative to “More Drainage Canals”**: use either this research‑focused mod *or* [[Gameplay] More Drainage Canals](https://mod.io/g/anno-117-pax-romana/m/more-drainage-channels#description); they are not designed to be combined in the same save.

## Optional Adjustments
You can manually tweak the strength of the research bonuses and the base width boost in this mod’s `assets.xml`:

- **Research pipe capacity bonuses (per upgrade):**  
  `Values/IrrigationUpgrade/PipeCapacityUpgrade/Value`

- **Research‑based maintenance reduction on Sluice Gates (per upgrade):**  
  `Values/MaintenanceUpgrade/MaintenanceFactorUpgrade/Value`

- **Drainage width (global base spread, same paths as in “More Drainage Canals”):**  
  `Values/IrrigationConfig/.../SourceSpreadMin`  
  `Values/IrrigationConfig/.../SourceSpreadMax`

If you prefer to control drainage purely via base values without research, you should use [[Gameplay] More Drainage Canals](https://mod.io/g/anno-117-pax-romana/m/more-drainage-channels#description) instead of this mod.

# DE – Beschreibung

**Erweitert den Forschungspfad um sechs neue Technologien rund um Schleusentore und Entwässerungskanäle, mit einer natürlichen Progression bei Kapazität und Unterhaltskosten – zusätzlich zu einer von Anfang an breiteren Entwässerungsreichweite.**  
Diese Mod fügt im Wirtschafts‑Forschungspfad einen kleinen Zweig hinzu, der die Schleusentore in Albion schrittweise verbessert, während die Grund‑Entwässerungsbreite global erhöht wird, sobald die Mod aktiv ist.

## Funktionen
- **Sechs neue Forschungsprojekte**, die sich vollständig um Schleusentore und Entwässerungskanäle drehen.
- **Mehr Entwässerungskanäle pro Schleusentor**: frühe Upgrades erhöhen die Rohr‑Kapazität, sodass jedes Schleusentor zusätzliche Kanäle unterstützen kann.
- **Größere Entwässerungsreichweite von Beginn an**: die grundlegende Reichweite der Entwässerungskanäle wird durch diese Mod global erhöht, unabhängig von den Forschungen.
- **Geringerer Unterhalt für das Schleusentor selbst**: spätere Upgrades reduzieren die Wartungskosten des Schleusentores, das alle angeschlossenen Kanäle antreibt.
- **Römischer Flair mit Begleittexten**: jede Stufe erhält einen kurzen Kommentar deines allgegenwärtigen römischen Baumeisters.
- **Alternative zu „More Drainage Canals“**: verwende entweder diese forschungsbasierte Mod *oder* [[Gameplay] More Drainage Canals](https://mod.io/g/anno-117-pax-romana/m/more-drainage-channels#description); beide sind nicht dafür gedacht, gleichzeitig im selben Spielstand genutzt zu werden.

## Optionale Anpassungen
In der `assets.xml` dieser Mod können die Stärke der Forschungsboni und der Basis‑Breiten‑Bonus manuell angepasst werden:

- **Rohr‑Kapazitätsboni durch Forschung (pro Upgrade):**  
  `Values/IrrigationUpgrade/PipeCapacityUpgrade/Value`

- **Wartungskosten‑Reduktion am Schleusentor durch Forschung (pro Upgrade):**  
  `Values/MaintenanceUpgrade/MaintenanceFactorUpgrade/Value`

- **Entwässerungsbreite (globale Basis‑Reichweite, wie bei „More Drainage Canals“):**  
  `Values/IrrigationConfig/.../SourceSpreadMin`  
  `Values/IrrigationConfig/.../SourceSpreadMax`

Wenn du die Entwässerung lieber ausschließlich über Basiswerte ohne Forschung steuern möchtest, solltest du stattdessen [[Gameplay] More Drainage Canals](https://mod.io/g/anno-117-pax-romana/m/more-drainage-channels#description) verwenden.
