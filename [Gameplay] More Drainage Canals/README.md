# EN – Description

**Increase how many drainage channels each Sluice Gate in Albion can support and slightly expand the drainage width.**  
This mod improves base and upgrade capacity while also increasing the tile spread range of each canal, allowing larger swamp areas to be drained with fewer Sluice Gates while keeping the gameplay balanced.

## Features
- **Base capacity increased** (75 → 100)
- **Upgrade capacity bonus increased** (+80 → +150)
- **Drainage channel width increased** (8 → 12 tiles per side)

## Optional Adjustments
You can manually edit the following values in `assets.xml`:

- **Base pipe capacity:**
  `Values/IrrigationSource/PipeFillCapacity`

- **Upgrade bonus:**
  `Values/IrrigationUpgrade/PipeCapacityUpgrade/Value`

- **Drain width (spread):**
  `Values/IrrigationConfig/.../SourceSpreadMin`
  `Values/IrrigationConfig/.../SourceSpreadMax`

- **Maintenance cost per unit length:**
  `Values/MeshGraphStyle/MaintenancePerUnitLength`
  *(Set to e.g. `0.0` if you want no maintenance cost.)*

---

## How to Install

### Manual Install
1. Download the mod via the Mod.io download button.
2. Extract the ZIP file into one of the following Anno 117 mod folders:

```
C:/Users/<Username>/Documents/Anno 117 - Pax Romana/mods
```

**OR**

```
C:/Program Files (x86)/Steam/steamapps/common/Anno 117 - Pax Romana/mods
```

**OR**

```
C:/Program Files (x86)/Ubisoft/Ubisoft Game Launcher/games/Anno 117 - Pax Romana/mods
```

3. Restart the game afterward.

# DE – Beschreibung

**Erhöht die Anzahl an Entwässerungskanälen pro Schleusentor in Albion und erweitert zusätzlich die Entwässerungsbreite.**
Diese Mod verbessert den Basis- und Upgrade-Wert und erhöht zudem die Reichweite des Kanals, damit größere Sumpfgebiete mit weniger Schleusentoren trockengelegt werden können – weiterhin mit fairem Balancing.

## Funktionen
- **Basis-Kapazität erhöht** (75 → 100)
- **Upgrade-Bonus verstärkt** (+80 → +150)
- **Entwässerungsbreite erhöht** (8 → 12 Felder pro Seite)

## Optionale Anpassungen
In der Datei `assets.xml` können folgende Werte angepasst werden:

- **Basis-Kapazität:**
  `Values/IrrigationSource/PipeFillCapacity`

- **Upgrade-Bonus:**
  `Values/IrrigationUpgrade/PipeCapacityUpgrade/Value`

- **Entwässerungsbreite (Spread):**
  `Values/IrrigationConfig/.../SourceSpreadMin`
  `Values/IrrigationConfig/.../SourceSpreadMax`

- **Wartungskosten pro Längeneinheit:**
  `Values/MeshGraphStyle/MaintenancePerUnitLength`
  *(Für keine Wartungskosten z.B. auf `0.0` setzen.)*
